## <a name="goal"></a>Goal
The objective of this project is to be able to create a Bank application with the typical functions: Create A account (Checking, Saving, Student, CreditCard), deposit money and transfer money.



## <a name="methodology"></a>Methodology
An Account Holder only can access information, deposit and transfers amount between accounts. Account holder can ask that its account should be delete from bank system.
A admin only can access (not modify) information account and update balance.
A thridParty only can transfer amount between accounts.
